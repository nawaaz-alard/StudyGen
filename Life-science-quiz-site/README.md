# Life Science Quiz Explorer

A gamified, interactive study aid for Grade 8 Natural Sciences (CAPS curriculum).

## Features
- **Study Mode**: Flip cards to learn key concepts.
- **Quiz Engine**: Drag-and-drop, matching, and text questions.
- **Interactive Diagrams**: Label the parts of Photosynthesis.
- **Gamification**: Earn XP, level up, and buy themes in the Bio-Shop.
- **Speed Blitz**: Test your reaction time with rapid-fire categorization.
- **PWA**: Installable app with offline support.

## How to Run Locally
1. Clone the repo.
2. Open `index.html` in your browser.
3. (Optional) Use a local server (e.g., Live Server) to test PWA features.

## Technologies
- HTML5, CSS3, Vanilla JavaScript
- No external frameworks or dependencies.
